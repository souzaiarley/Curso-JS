Sintaxe JavaScript

1- Variáveis podem ser declaradas pelas palavras-chave "var", "let" e "const".
ex:
    var a=1;
    var b=5;
    var c=a+b;

2- O JavaScript é case-sensitive.
ex:
    var PESSOA = "Dimitri";
    var pessoa = "Teixeira";

    -> As duas variáveis são diferentes.

3- Dicas:
    -> Recomenda-se o uso de CamelCase.
